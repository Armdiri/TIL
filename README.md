# TIL (Today I Learned)

개인 학습 내용을 기록하는 저장소입니다.

## 목적
- 매일 배운 내용을 정리하고 기록
- 지식의 체계적 정리
- 학습 내용의 공유

## 사용 방법
1. 각 주제별로 디렉토리를 생성하여 내용을 정리
2. 마크다운 형식으로 작성
3. 정기적으로 커밋하여 기록

## 디렉토리 구조
```
TIL/
├── programming/     # 프로그래밍 관련 학습 내용
├── algorithm/      # 알고리즘 관련 학습 내용
├── database/       # 데이터베이스 관련 학습 내용
└── etc/            # 기타 학습 내용
```
